## 3. APORTES ADICIONALES

3.1. EL PARTICIPANTE puede realizar aportes adicionales en cualquier momento.

3.2. Cada aporte se calculara como porcentaje del valor del portafolio al momento de recibirlo:

```
Nuevo % = (Aporte / Valor Total del Portafolio) * 100
```

3.3. El nuevo porcentaje se sumara al porcentaje existente de EL PARTICIPANTE.

3.4. EL ADMINISTRADOR confirmara por escrito (WhatsApp) cada aporte recibido, indicando:

## 4. RETIROS

4.1. EL PARTICIPANTE puede solicitar retiros en cualquier momento, sin restricciones.

4.2. El monto del retiro se calculara segun el valor actual del portafolio:

```
Monto a Retirar = (Porcentaje de Participacion) * (Valor Actual del Portafolio)
```

4.3. EL ADMINISTRADOR procesara el retiro dentro de un plazo razonable, sujeto a la liquidez disponible en el portafolio.

4.4. En caso de retiro total, el porcentaje de participacion de EL PARTICIPANTE se reducira a 0%.
