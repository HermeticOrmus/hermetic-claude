# Quick Fix

I'm having an issue. Help me troubleshoot and fix it quickly.

## Quick Diagnostic Questions

1. **What's the problem?**
   - What's not working?
   - What did you expect to happen?
   - What actually happened?

2. **When did it start?**
   - Just now?
   - After an update?
   - After a change?
   - Always been broken?

3. **Error messages?**
   - Copy exact error message
   - Check logs for errors
   - Screenshots if helpful

4. **What changed recently?**
   - New software installed?
   - Config changes?
   - System updates?
   - Code changes?

## Common Fixes

**"It doesn't work"**:
1. Define "doesn't work" precisely
2. Check logs/errors
3. Verify prerequisites
4. Test minimal case

**"It worked before"**:
1. What changed?
2. Revert recent changes
3. Compare with working version
4. Check git diff

**"Permission denied"**:
1. Check file permissions
2. Verify user/group
3. Check sudo requirements
4. Review path accessibility

**"Command not found"**:
1. Is it installed?
2. Is it in PATH?
3. Check spelling
4. Try absolute path

**"Port already in use"**:
1. Find process using port
2. Kill it or use different port
3. Check for zombie processes
