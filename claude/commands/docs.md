# Documentation

Please help me create or improve documentation for this project or code.

Documentation types:
1. **README.md**: Project overview, setup, usage
2. **CLAUDE.md**: Context for Claude Code
3. **API Documentation**: Function/class/endpoint docs
4. **User Guide**: How to use the project
5. **Developer Guide**: How to contribute/develop
6. **Architecture**: System design and decisions
7. **Changelog**: Version history and changes

Good documentation should:
- Be clear and concise
- Include examples
- Cover common use cases
- Explain "why" not just "how"
- Be easy to maintain
- Be discoverable

Help me create documentation that makes the project accessible and maintainable.
