# Code Review

Please perform a thorough code review of the current changes or specified files.

Review checklist:
1. **Code Quality**: Logic, structure, readability, maintainability
2. **Best Practices**: Following language/framework conventions
3. **Performance**: Potential bottlenecks or optimizations
4. **Security**: Common vulnerabilities or security issues
5. **Testing**: Test coverage and quality
6. **Documentation**: Comments, docs, and clarity
7. **Error Handling**: Proper error management
8. **Edge Cases**: Potential bugs or missing scenarios

Provide specific, actionable feedback with examples where helpful. Be constructive and educational.
