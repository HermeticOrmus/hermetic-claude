# Deploy Application

Help me deploy this application safely and correctly.

## Pre-Deployment Checklist

1. **Verify Tests**
   - Run full test suite
   - Ensure all tests pass
   - Check test coverage

2. **Check Version**
   - Verify version number is bumped
   - Check CHANGELOG is updated
   - Ensure git tags are correct

3. **Build Verification**
   - Run production build
   - Check for build errors/warnings
   - Verify build artifacts

4. **Environment Check**
   - Confirm deployment environment
   - Check environment variables
   - Verify credentials/secrets

5. **Dependency Review**
   - Check for security vulnerabilities
   - Ensure dependencies are up to date
   - Verify lock files are committed

## Deployment Process

1. Create deployment plan
2. Execute pre-deployment checks
3. Build for production
4. Deploy to target environment
5. Run smoke tests
6. Monitor for errors
7. Verify deployment success

## Post-Deployment

- Monitor logs and metrics
- Verify critical functionality
- Check performance
- Document deployment
- Notify stakeholders if needed

Guide me through each step, asking for confirmation before proceeding.
