## Maintenance Practices

**Weekly**: Review active projects, archive completed work
**Monthly**: Clean up Dev/ directory, update documentation
**Quarterly**: Major organization review, restructure if needed

Only when existing categories don't fit. Prefer using existing structure to avoid fragmentation.
